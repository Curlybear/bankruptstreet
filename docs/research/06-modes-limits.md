# Modes, Rule Sets, Numeric Limits

## Easy rules vs Standard rules **[V]**

| Aspect | Easy | Standard |
|---|---|---|
| Stocks | none | full market |
| Districts | none (adjacent-shop expansion instead) | full district system |
| Vacant plots / buildings | none | yes |
| Venture cards | 105 (no stock/district cards) | all 128 |
| Slurpodrome rewards | doubled | normal |

Easy is the on-ramp; Standard is the real game (and what Bankrupt Street
models).

## Tour mode

- Fixed sequence of boards vs. preset opponents; finishing 2nd or better
  typically unlocks the next board/character.
- **Game ends at the first bankruptcy** in Tour.
- Completing both franchise tours unlocks special boards.

## Custom mode

- Target net worth configurable **6,000–999,000G** (1,000G steps).
- **Bankruptcy limit** configurable: end after one, two, or last-man-standing.
- Easy/Standard selectable.
